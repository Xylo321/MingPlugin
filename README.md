# MingPlugin

做WEB爬虫而开发的工具。对研究浏览器反爬有一定的方便。

![](http://serv_pro:3000/kael/MingPlugin/raw/master/look.jpg)

可以用它来：
* 设置代理
* 设置请求头
* 拦截参数
* 清除cookie
* 清除历史	
* 16进制解密
* 16进制加密
* base64解码
* base64编码
* url编码
* url解码
* 时间戳转换
* 正则表达式测试
* json格式化
* 图片转base64
* JSON格式化
